# 📱 Mensagens Melhoradas - Telegram

## ✅ O Que Foi Melhorado

As mensagens dos alertas foram reformuladas para ficarem mais profissionais e claras.

---

## 📨 Novas Mensagens

### 1. Movimento Detectado

```
🚶 ALERTA: Movimento Detectado

📊 Confiança: 60%
🕐 Horário: 22:15:30

ℹ️ Atividade detectada no ambiente monitorado
```

**Quando aparece**: Quando alguém se movimenta no ambiente

---

### 2. Presença Detectada

```
👤 ALERTA: Presença Detectada

📊 Confiança: 75%
🕐 Horário: 22:15:45

ℹ️ Pessoa presente no ambiente
```

**Quando aparece**: Quando detecta alguém parado no local

---

### 3. Queda Detectada

```
🚨 ALERTA: Queda Detectada

📊 Confiança: 95%
🕐 Horário: 22:16:00

⚠️ Ação recomendada:
Verifique imediatamente o local

📋 Detalhes:
  • Taxa de mudança: 15.5 dB/s
```

**Quando aparece**: Movimento brusco ou queda de sinal >15 dBm

---

### 4. Inatividade Prolongada

```
⏰ ALERTA: Inatividade Prolongada

📊 Confiança: 85%
🕐 Horário: 22:16:30

⚠️ Sem movimento há mais de 30 segundos

📋 Detalhes:
  • Duração: 2.5 min
```

**Quando aparece**: Sem movimento por mais de 30 segundos

---

## 📋 Características das Novas Mensagens

1. **Título claro**: "ALERTA: [Tipo de Evento]"
2. **Informações organizadas**: Confiança e horário em linhas separadas
3. **Contexto adicional**: Explicação do que significa cada alerta
4. **Ações recomendadas**: Para eventos críticos (quedas)
5. **Detalhes técnicos**: Apenas informações relevantes
6. **Emojis apropriados**: Facilitam identificação rápida
