## 🚀 Comandos para Iniciar

### Iniciar Backend
```bash
cd backend
uvicorn app.main:app --reload
```

### Iniciar Frontend
```bash
cd frontend
npm run dev
```

## 📱 Configurar Telegram (5 minutos)

1. Abra o Telegram
2. Procure: `@BotFather`
3. Digite: `/newbot`
4. Siga as instruções
5. **Copie o TOKEN**
6. Procure: `@userinfobot` para pegar seu Chat ID

**Guia completo**: `backend/docs/GUIA_TELEGRAM_BOT.md`
